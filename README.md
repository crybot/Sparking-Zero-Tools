# Sparking-Zero-Tools

Quality-of-life mod for **Dragon Ball Sparking! Zero** (UE 5.1.1)

## How To Use

Download the latest release from the [Releases page](https://github.com/crybot/Sparking-Zero-Tools/releases) and unzip the archive.
Start the game, then run the injector executable.
The UI menu will only appear once you enter the gameâ€™s main menu.

## Features

* **Network latency monitoring**: shows realtime ping, running averages, jitter, packet loss, and min/max values between players.
* **FPS monitoring**: displays the current FPS for both players along with live FPS graphs.
* **Remaining time widget toggle** to enable or disable the in-game timer display.
* **Smash Level meter** that displays combo momentum for both players.
* **DP Battle win condition score**, computed from character HP percentages and Destruction Points.
* **Custom map override** for ranked matches, letting you select from several pre-existing maps.

## Notes

* Custom maps only work if you are the **host**; otherwise the match will disconnect.
* Do not use custom map replacement in private matches.
* **Only works with game version 2016.015.003.009.008**
